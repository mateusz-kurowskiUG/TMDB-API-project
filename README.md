# TMDB-API-project

This project has been created for the purposes of Frontend Development and Databases classes.
## Technologies

- Typescript -> Express.js API
- React -> Next.js
- Neo4j
- TailwindCSS
  
## Deployment

To start the project:

In `backend` and `frontend` directories:

- `pnpm install`
- `pnpm start` / `pnpm dev`
